# 📐 Calculadora de MRUV Online

## 📖 Descripción
Este proyecto consiste en una **calculadora web de Movimiento Rectilíneo Uniformemente Variado (MRUV)** que permite resolver problemas de cinemática de forma **rápida e interactiva**.

La herramienta está diseñada para ayudar a estudiantes a **verificar resultados y comprender mejor las fórmulas del MRUV**.

---

## ⚙️ Funcionalidades

La calculadora permite resolver diferentes ecuaciones del **Movimiento Rectilíneo Uniformemente Variado**, como:

- 🔹 Cálculo de **velocidad final**
- 🔹 Cálculo de **velocidad inicial**
- 🔹 Cálculo de **aceleración**
- 🔹 Cálculo de **tiempo**
- 🔹 Cálculo de **desplazamiento o posición**

✅ El sistema calcula automáticamente el resultado cuando el usuario ingresa los valores conocidos.

---

## 🖥️ Tecnologías utilizadas

Este proyecto fue desarrollado utilizando tecnologías web básicas:

- **HTML** → estructura de la página  
- **CSS** → diseño de la interfaz  
- **JavaScript** → lógica de cálculo de las fórmulas  

---

## 🚀 Cómo usar la calculadora

1️⃣ Abrir la página web de la calculadora.  
2️⃣ Ingresar los valores conocidos del problema.  
3️⃣ Presionar el botón **Calcular**.  
4️⃣ La calculadora mostrará automáticamente el resultado.

---
